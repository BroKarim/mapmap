First, install depts (Do this only once):
```bash
npm install
# or
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

### How to remove those strict linting rules

You can adjust the settings in ```eslint.json``` and ```tsconfig.json```.
